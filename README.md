# Проект Muse — Переверстка на Tailwind CSS

> **Главная точка входа в документацию проекта**

## О проекте

Переверстка сайта **www.muse.ooo** с Bootstrap 3 на Tailwind CSS.

| Показатель | Значение |
|------------|----------|
| Всего страниц | 59 |
| Готово | 55 (93%) |
| Ожидает | 4 |

---

## Быстрый старт

### Установка и запуск

```bash
npm install         # Установка зависимостей
npm run build       # Сборка CSS (с отслеживанием)
npm run preview     # Просмотр в браузере
```

### Быстрый деплой

Запустите **`БЫСТРЫЙ_ДЕПЛОЙ.bat`** — он автоматически соберёт и опубликует.

**Сайт на Vercel:** https://muse-liard-one.vercel.app/

## Структура проекта

```
tailwind-project/
├── src/html/              # HTML страницы (55 файлов)
│   ├── js/nav.js          # Page Navigator + Back to Top
│   ├── blog/              # Блог (20 страниц)
│   ├── info/              # Info (3 страницы)
│   └── portret-na-zakaz/  # Портреты (style/, object/)
├── docs/                  # Документация
└── tailwind.config.js     # Конфигурация
```

## Технические особенности

- **CSS:** Tailwind CDN для разработки, output.css для продакшена
- **Скрипты:** `js/nav.js` (Page Navigator, Back to Top) + inline скрипты
- **Контейнер:** 1170px, padding 15px
- **Брейкпоинты:** sm=640, md=768, lg=1024, xl=1280
