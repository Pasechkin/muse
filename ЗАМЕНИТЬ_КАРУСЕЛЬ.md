# Замена кода карусели в index.html

## Текущая проблема
Forced reflow: 873ms из-за чтения геометрических свойств

## Решение
Заменить весь блок кода карусели (строки 869-1148) на оптимизированную версию

## Файлы
- `CAROUSEL_OPTIMIZED.js` - оптимизированный код
- `test-carousel.html` - тестовая страница для сравнения

## Инструкция
1. Откройте `index.html` 
2. Найдите строку `// Прокрутка каруселей колесиком мыши на ПК (оптимизированная версия)`
3. Замените весь блок до строки `});` (закрывающая скобка forEach)
4. Используйте код из `CAROUSEL_OPTIMIZED.js`
