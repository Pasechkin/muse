═══════════════════════════════════════════════════════════════
  ЧТО ПРОИСХОДИТ СЕЙЧАС:
═══════════════════════════════════════════════════════════════

1. 🔄 Vercel клонирует ваш репозиторий из GitHub
   → https://github.com/Pasechkin/muse

2. 📦 Устанавливает зависимости
   → npm install
   → Устанавливает Tailwind CSS, PostCSS и другие пакеты

3. 🏗️ Собирает проект
   → npm run build:once
   → Компилирует CSS из Tailwind
   → npm run copy-css
   → Копирует CSS в папку src/html

4. 🚀 Деплоит сайт
   → Загружает файлы из папки src/html
   → Настраивает CDN и HTTPS

5. ✅ Готово! (займет 2-5 минут)

❌ ОШИБКА ПРИ СБОРКЕ
───────────────────────────────────────────────────────────────
1. Откройте логи деплоя в Vercel
2. Проверьте ошибки (обычно красным цветом)
3. Частые проблемы:
   • Не хватает зависимостей → проверьте package.json
   • Ошибки в CSS → проверьте src/input.css
   • Неправильный путь к файлам → проверьте vercel.json

💡 РЕШЕНИЕ:
   → Проверьте, что все файлы загружены на GitHub
   → Убедитесь, что package.json корректен
   → Проверьте логи ошибок - там будет указана причина


❌ САЙТ НЕ ОТОБРАЖАЕТСЯ ПРАВИЛЬНО
───────────────────────────────────────────────────────────────
1. Проверьте, что CSS подключен:
   → Откройте исходный код страницы (Ctrl+U)
   → Найдите: <link rel="stylesheet" href="/css/output.css">
   → Если нет - проверьте путь в HTML

2. Проверьте консоль браузера (F12):
   → Откройте вкладку Console
   → Проверьте ошибки 404 (не найдены файлы)
   → Проверьте ошибки CSS

💡 РЕШЕНИЕ:
   → Убедитесь, что buildCommand в vercel.json правильный
   → Проверьте, что outputDirectory указан как "src/html"
   → Проверьте пути к CSS в index_tailwind.html


❌ САЙТ ПУСТОЙ ИЛИ НЕ ТОТ ФАЙЛ
───────────────────────────────────────────────────────────────
1. Проверьте outputDirectory в vercel.json:
   → Должно быть: "src/html"
   
2. Проверьте, какой файл открывается:
   → Главная страница должна быть: src/html/index_tailwind.html
   → Но Vercel ищет index.html
   
💡 РЕШЕНИЕ:
   → Скопируйте index_tailwind.html в index.html
   → Или настройте в vercel.json, какой файл открывать
