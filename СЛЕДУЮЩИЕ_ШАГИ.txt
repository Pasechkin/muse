   (скопирует index_tailwind.html → index.html с инлайновым CSS)

   (отправит на GitHub, Vercel обновится автоматически)

Вариант 2 (если редактируете только index.html):
1. Скопируйте index.html → index_tailwind.html
2. В index_tailwind.html замените:
   <link rel="stylesheet" href="css/output.css">
   на инлайновый CSS (можно использовать скрипт в будущем)
3. Запустите: СИНХРОНИЗИРОВАТЬ_ФАЙЛЫ.bat
4. Запустите: БЫСТРЫЙ_ДЕПЛОЙ.bat

💡 РЕКОМЕНДАЦИЯ:

Для простоты можете работать ТОЛЬКО с index.html локально.

Когда нужно загрузить на сервер:
1. Откройте index.html и index_tailwind.html в редакторе
2. Скопируйте всё содержимое body из index.html в index_tailwind.html
3. Запустите: СИНХРОНИЗИРОВАТЬ_ФАЙЛЫ.bat
4. Запустите: БЫСТРЫЙ_ДЕПЛОЙ.bat

Или используйте локальный сервер для разработки:
- Запустите: npm run preview
- Откройте: http://localhost:3000
- Редактируйте файлы, видите изменения сразу
