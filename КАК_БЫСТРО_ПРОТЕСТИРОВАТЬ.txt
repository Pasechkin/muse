========================================
КАК БЫСТРО ПРОТЕСТИРОВАТЬ ИЗМЕНЕНИЯ
========================================

Vercel автоматически обновляет сайт при каждом изменении в GitHub.

БЫСТРЫЙ СПОСОБ (1-3 минуты):
-----------------------------
1. Дважды кликните на файл: БЫСТРЫЙ_ДЕПЛОЙ.bat
2. Дождитесь завершения (скрипт сам всё сделает)
3. Подождите 1-3 минуты пока Vercel задеплоит
4. Обновите страницу: https://muse-liard-one.vercel.app/

ЧТО ДЕЛАЕТ СКРИПТ:
------------------
✓ Собирает CSS (npm run build:once)
✓ Копирует CSS в нужную папку
✓ Добавляет все изменения в Git
✓ Создает коммит
✓ Отправляет на GitHub
✓ Vercel автоматически запускает деплой

РУЧНОЙ СПОСОБ:
--------------
Если скрипт не работает, выполните вручную:

1. Откройте PowerShell в папке tailwind-project
2. Выполните команды:

   npm run build:once
   npm run copy-css
   git add .
   git commit -m "Оптимизация"
   git push

3. Подождите 1-3 минуты
4. Проверьте сайт: https://muse-liard-one.vercel.app/

ПРОВЕРКА СТАТУСА ДЕПЛОЯ:
-------------------------
1. Откройте: https://vercel.com/dashboard
2. Найдите проект "muse-liard-one"
3. Посмотрите статус последнего деплоя

ВРЕМЯ ОЖИДАНИЯ:
---------------
- Обычно: 1-3 минуты
- При большой нагрузке: до 5 минут
- Можно обновлять страницу каждые 30 секунд

ПОСЛЕ ДЕПЛОЯ:
-------------
✓ Обновите страницу (Ctrl+F5 для полной перезагрузки)
✓ Проверьте изменения
✓ Запустите Lighthouse для проверки оптимизаций
