  ВАЖНО: После установки Git
═══════════════════════════════════════════════════════════════

После установки Git для Windows вам нужно:

1. ЗАКРЫТЬ все окна PowerShell/Terminal/Command Prompt
2. ЗАКРЫТЬ все окна VS Code/Cursor (если открыты)
3. ОТКРЫТЬ новое окно PowerShell или Terminal
4. Запустить скрипт: ЗАГРУЗИТЬ_НА_GITHUB_УЛУЧШЕННЫЙ.bat

Почему это важно?
───────────────────────────────────────────────────────────────
Git устанавливается, но PowerShell не видит его, пока не будет
перезапущен. После перезапуска PowerShell Git появится в PATH.

═══════════════════════════════════════════════════════════════

БЫСТРЫЙ СПОСОБ:
───────────────────────────────────────────────────────────────
1. Дважды кликните на файл: ЗАГРУЗИТЬ_НА_GITHUB_УЛУЧШЕННЫЙ.bat
2. Скрипт сам найдет Git, даже если PowerShell его не видит
3. Следуйте инструкциям на экране
