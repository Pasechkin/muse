Паттерн инпутов: ah-50 → ah-25 при фокусе

По умолчанию: bg ah-50, border ah-200, text ah-950, placeholder ah-700
При :focus: bg ah-25, outline ah-700, border transparent
Почему: ah-50 даёт лёгкий тон, показывающий «зона ввода». При фокусе переход на ah-25 создаёт ощущение «стеклянной чистоты» — как будто поверхность стала ярче.

WCAG: ah-950 на ah-50 = 16.5:1 (AAA ✅) · ah-700 на ah-50 = 4.37:1 (AA для крупного текста ✅)
