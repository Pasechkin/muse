Design System Compare

Прямое сравнение O и Muse

Одинаковые сценарии показаны слева и справа. Без длинных объяснений: только как выглядит заголовок, обычный текст, текст на тёмном фоне, отступы и кнопки.

Template O

Oatmeal / olive system

Role-first иерархия из stable-примитивов Heading, Subheading, Text, Container. Форма мягкая, круглые кнопки, serif-display + neutral olive palette.

Fonts Instrument Serif для display, Inter для текста и UI.
Container max-w-2xl → md:max-w-3xl → lg:max-w-7xl, px-6 → lg:px-10.
Rhythm Частые интервалы: gap-6, gap-16, gap-32, section py-16.

Hero / H1

Очень большой, мягкий, почти editorial.

2. Заголовок секции

3. Обычный текст

Body

O использует спокойный текст с большим воздухом между строк. Он кажется мягким и достаточно “премиальным”, даже когда это обычный абзац.

Моб: 16px Десктоп: 16px

Heading on dark background

7. Кратко

Суть O

Заголовки Крупные и очень выразительные
Обычный текст Не мелкий, с большим воздухом
Кнопки Круглые и мягкие
Muse

Amber Harmonized system

Display-serif + PT Sans, ink-система на amber-палитре, жёстче CTA-геометрия и более прикладная продуктовая структура: hero, content sections, CTA, utility classes.

Fonts Playfair Display для heading-слоя, PT Sans для текста, кнопок и UI.
Container Один container: max-width 1170px, padding-inline 16px.
Rhythm Основные секции py-16 lg:py-24, CTA py-24 md:py-32, footer py-8 lg:py-12.
