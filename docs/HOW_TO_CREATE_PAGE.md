## Выбор шаблона

Выберите подходящий шаблон в зависимости от типа страницы:

### Base Template (Базовый шаблон)
**Файл:** `src/templates/base.html`

**Использование:** Для страниц с уникальной структурой

### Category Template (Шаблон категории)
**Файл:** `src/templates/category.html`

**Использование:** Для страниц категорий товаров/услуг
- Примеры: "Печать на холсте", "Портреты на заказ"

### Product Template (Шаблон товара)
**Файл:** `src/templates/product.html`

**Использование:** Для страниц отдельных товаров/стилей
- Примеры: "Портрет маслом", "Фото на холсте"

### Info Template (Информационный шаблон)
**Файл:** `src/templates/info.html`

**Использование:** Для информационных страниц
- Примеры: "О нас", "Доставка", "Контакты"

## Создание страницы

### Шаг 1: Определите структуру папок

Создайте папку для страницы по структуре URL:

```
src/html/
  ├── pechat/              # Категория "Печать"
  │   └── index.html
  ├── portret-na-zakaz/    # Категория "Портреты"
  │   └── index.html
  └── info/                # Информационные страницы
      ├── dostavka/
      │   └── index.html
      └── kontakty/
          └── index.html
```

### Шаг 2: Скопируйте шаблон

1. Откройте нужный шаблон (например, `src/templates/category.html`)
2. Скопируйте весь код
3. Создайте новый файл в нужной папке (например, `src/html/pechat/index.html`)
4. Вставьте скопированный код

### Шаг 3: Настройте базовую информацию

В `<head>` замените:

```html
<title>Название страницы - Muse</title>
<meta name="description" content="Описание страницы">
```

## Заполнение контента

### Шаг 1: Вставьте Header и Footer

Найдите комментарии в шаблоне:
```html
<!-- Header -->
<!-- Вставьте сюда компонент header/header.html -->
```

Откройте `src/components/header/header.html`, скопируйте весь код и вставьте вместо комментария.

Аналогично для Footer:
```html
<!-- Footer -->
<!-- Вставьте сюда компонент footer/footer.html -->
```

### Шаг 2: Настройте Hero секцию

Для страниц категорий и товаров:

1. Замените изображение:
```html
<img src="https://muse.ooo/upload/img/your-image.webp" alt="Описание">
```

2. Замените заголовки:
```html
<h1 class="h1 text-white mb-4">Ваш заголовок</h1>
<p class="lead text-white">Ваше описание</p>
```

### Шаг 3: Добавьте контент

#### Для категорий (Category Template):

1. **Описание категории:**
```html
<h2 class="h2 text-dark mb-4">О категории</h2>
<p class="text-body">Ваше описание категории.</p>
```

2. **Карточки товаров:**
   - Откройте `src/components/cards/product-card.html`
   - Скопируйте нужный вариант
   - Вставьте в сетку товаров
   - Замените изображения, ссылки, названия

#### Для товаров (Product Template):

1. **Изображения товара:**
```html
<img src="https://muse.ooo/upload/img/product-main.webp" alt="Описание">
```

2. **Описание товара:**
```html
<h2 class="h2 text-dark mb-4">Описание</h2>
<p class="text-body">Ваше описание товара.</p>
```

3. **Похожие товары:**
   - Используйте компонент `product-card.html`

#### Для информационных страниц (Info Template):

1. **Контент страницы:**
```html
<h2 class="h2 text-dark mb-4">Заголовок раздела</h2>
<p class="text-body mb-4">Ваш текст.</p>
```

2. **Списки:**
```html
<ul class="list-disc list-inside text-body mb-4 space-y-2">
  <li>Пункт 1</li>
  <li>Пункт 2</li>
</ul>
```

### Шаг 4: Добавьте дополнительные секции (опционально)

#### Секция преимуществ:
1. Откройте `src/components/sections/advantages.html`
2. Скопируйте код
3. Вставьте в нужное место
4. Добавьте карточки преимуществ из `feature-card.html`

#### CTA секция:
1. Откройте `src/components/sections/cta.html`
2. Скопируйте код
3. Вставьте перед Footer
4. Настройте текст и кнопки

#### Промо-баннер:
1. Откройте `src/components/sections/promo-banner.html`
2. Скопируйте код
3. Вставьте после Hero секции (если нужен)

## Проверка

### Шаг 1: Проверьте структуру

Убедитесь, что:
- ✅ Header вставлен в начале body
- ✅ Footer вставлен перед закрывающим `</body>`
- ✅ Все компоненты скопированы полностью
- ✅ Пути к CSS правильные (относительно вашей папки)

### Шаг 2: Проверьте контент

- ✅ Все тексты заменены
- ✅ Все ссылки работают
- ✅ Все изображения загружаются
- ✅ Мета-теги заполнены

### Шаг 3: Проверьте адаптивность

Откройте страницу в браузере и проверьте:
- ✅ На десктопе (широкий экран)
- ✅ На планшете (средний экран)
- ✅ На мобильном (узкий экран)

Используйте инструменты разработчика браузера (F12) для проверки разных размеров экрана.

### Шаг 4: Проверьте производительность

- ✅ Изображения оптимизированы (WebP формат)
- ✅ Используется `loading="lazy"` для изображений ниже первого экрана
- ✅ CSS подключен правильно

## Примеры

### Пример 1: Создание страницы категории "Печать на холсте"

1. Создайте папку: `src/html/pechat/`
2. Скопируйте `src/templates/category.html` в `src/html/pechat/index.html`
3. Замените заголовок: "Печать на холсте - Muse"
4. Вставьте Header и Footer
5. Настройте Hero секцию
6. Добавьте 4 карточки товаров из `product-card.html`
7. Добавьте CTA секцию
8. Проверьте результат

### Пример 2: Создание информационной страницы "Доставка"

1. Создайте папку: `src/html/info/dostavka/`
2. Скопируйте `src/templates/info.html` в `src/html/info/dostavka/index.html`
3. Замените заголовок: "Доставка - Muse"
4. Вставьте Header и Footer
5. Настройте Hero секцию (или уберите, если не нужна)
6. Заполните контент страницы
7. Добавьте списки, если нужны
8. Проверьте результат

## Частые ошибки

### Ошибка 1: Неправильный путь к CSS

**Проблема:** CSS не загружается

**Решение:** Проверьте относительный путь к CSS файлу:
- Если страница в `src/html/pechat/index.html`, путь: `../css/output.css`
- Если страница в `src/html/info/dostavka/index.html`, путь: `../../css/output.css`

### Ошибка 2: Компоненты не отображаются

**Проблема:** Компонент скопирован не полностью

**Решение:** Убедитесь, что скопированы все открывающие и закрывающие теги

### Ошибка 3: Мобильное меню не работает

**Проблема:** Скрипт не загружается

**Решение:** Убедитесь, что в конце страницы есть скрипт для загрузки `@tailwindplus/elements`
