# Начало редактирования новой страницы

Инструкция для работы с ИИ-агентом в Cursor при создании/редактировании страниц проекта Muse.

---

## Шаг 1: Выбор режима

| Режим | Когда использовать |
|-------|-------------------|
| **Ask (Plan)** | В начале — для обсуждения структуры и уточнения деталей |
| **Agent** | После согласования плана — для выполнения работы |

**Рекомендация:** Начните с **Ask**, согласуйте план, затем переключитесь на **Agent**.

---

## Шаг 3: Вопросы для уточнения (режим Ask)

Перед началом работы уточните:

1. **Какие блоки/секции есть на странице?**
   - Чтобы не пропустить секции

2. **Есть ли видео на странице?**
   - Нужна оптимизация (preload, media)

3. **Какой Hero на странице?**
   - С видео, с изображением, без Hero

4. **Есть ли специфичные компоненты?**
   - Калькулятор, слайдеры, формы, карусели

5. **Количество элементов в секциях?**
   - Сколько примеров, FAQ-вопросов, преимуществ

---

## Шаг 4: Переключение на Agent

После согласования плана напишите:

```
План согласован. Начинай работу.
```

Или просто переключитесь на режим **Agent** в интерфейсе Cursor.

---

## Шаг 5: Проверка после завершения

Попросите агента проверить по чек-листу:

```
Проверь страницу по чек-листу из AI_INSTRUCTIONS.md:
- Контент совпадает с оригиналом
- Количество элементов совпадает
- Изображения имеют все атрибуты
- Видео оптимизировано (preload="none")
- Формы имеют label
- Ссылки имеют underline
```

Затем попросите проверить валидацию:

```
Проверь страницу через W3C Validator:
https://validator.w3.org/nu/?doc=https://muse-liard-one.vercel.app/[путь-к-странице]
```

## Частые ошибки (избегайте)

❌ **Не писать:**
- "Сделай страницу" (слишком общо)
- "Возьми контент откуда-нибудь" (непонятно откуда)
- "Сделай как обычно" (нет конкретики)

✅ **Писать:**
- Конкретный URL источника контента
- Ссылку на эталон
- Ключевые правила (не менять контент, предупреждать об урезании)
