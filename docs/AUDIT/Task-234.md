Задание 2: Постраничное сравнение контента с оригиналом muse.ooo
Цель: Проверить, что контент каждой страницы (тексты, количество элементов, структура секций) совпадает с оригинальным сайтом www.muse.ooo.

Инструкция:

Сравни контент страницы [src/html/ИМЯ_ФАЙЛА.html] с оригинальной страницей на muse.ooo.

**Что проверить:**

1. **Тексты и заголовки:**
   - H1 совпадает с оригиналом
   - Все текстовые блоки присутствуют и идентичны
   - Нет сокращений или перефразированного текста
   - Нет плейсхолдеров типа "Lorem ipsum"

2. **Количество элементов:**
   - FAQ: количество вопросов-ответов совпадает
   - Преимущества: количество карточек совпадает
   - Примеры работ: количество изображений в карусели совпадает
   - Стили портретов: количество категорий совпадает

3. **Структура секций:**
   - Порядок секций на странице совпадает с оригиналом
   - Все секции присутствуют (ничего не пропущено)

4. **Alt и Title изображений:**
   - Alt тексты совпадают с оригиналом
   - Title атрибуты сохранены (если были на оригинале)
