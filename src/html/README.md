# Верхнее меню для сайта Muse

## Описание

Современное адаптивное верхнее меню для сайта студии Muse, созданное с использованием Tailwind CSS.

## Файлы

- `header.html` - Полная HTML страница с примером использования меню
- `navbar.html` - Компонент меню для интеграции в существующий сайт

## Особенности

- ✅ Адаптивный дизайн (работает на всех устройствах)
- ✅ Sticky позиционирование (меню остается видимым при прокрутке)
- ✅ Мобильное меню с анимацией
- ✅ Плавные переходы и эффекты наведения
- ✅ Соответствие стилю темы Stack
- ✅ Использование цветов бренда (#4a90e2)

## Использование

### Вариант 1: Интеграция в PHP (Bitrix)

Скопируйте содержимое файла `navbar.html` в ваш шаблон Bitrix, например в `header.php`:

```php
<?php
// Подключение CSS
use Bitrix\Main\Page\Asset;
Asset::getInstance()->addCss('/tailwind-project/src/css/output.css');
?>

<!-- Вставьте содержимое navbar.html здесь -->
```

### Вариант 2: Использование как отдельный компонент

1. Убедитесь, что CSS скомпилирован:
   ```bash
   npm run build:once
   ```

2. Подключите CSS в `<head>`:
   ```html
   <link rel="stylesheet" href="/tailwind-project/src/css/output.css">
   ```

3. Вставьте содержимое `navbar.html` в нужное место вашего шаблона

## Настройка

### Изменение пунктов меню

Отредактируйте ссылки в файле `navbar.html`:

```html
<a href="/ваш-раздел/" class="...">Название раздела</a>
```

### Изменение логотипа

Замените текст "Muse" на изображение логотипа:

```html
<a href="/" class="flex items-center">
    <img src="/img/logo.png" alt="Muse" class="h-10">
</a>
```

### Изменение цветов

Цвета настраиваются в `tailwind.config.js`:

- Основной цвет: `primary: '#4A90E2'`
- Темный цвет: `dark: '#252525'`
- Цвет текста: `body: '#666666'`

## Структура меню

- **Логотип** - ссылка на главную страницу
- **Портреты** - раздел портретов
- **Печать** - раздел печати на холсте
- **Блог** - блог студии
- **О нас** - информация о студии
- **Контакты** - контактная информация
- **Заказать** - кнопка для оформления заказа

## Адаптивность

- **Desktop (lg и выше)**: Горизонтальное меню со всеми пунктами
- **Mobile (меньше lg)**: Гамбургер-меню с выпадающим списком

## Поддержка браузеров

Меню работает во всех современных браузерах:
- Chrome/Edge (последние версии)
- Firefox (последние версии)
- Safari (последние версии)
- Mobile браузеры (iOS Safari, Chrome Mobile)

## Дополнительные возможности

Для добавления подменю или выпадающих списков можно расширить компонент, добавив соответствующие элементы и JavaScript логику.
