Набор иконок (Полный)

Все иконки были очищены и приведены к единому формату для Tailwind CSS.

Как использовать

1. Размер и Цвет: Теперь управляются через классы .icon-lg (большая) и .icon-sm (маленькая) в блоке <style>.

2. Индивидуальный цвет: Для маленьких иконок цвет можно переопределить классом, например text-red-500.

3. Обводка (Stroke): У этих иконок нет обводки, они состоят из залитых фигур. Толщину линии (stroke-width) изменить нельзя. Если нужна такая функция, лучше использовать наборы линейных иконок, например, Heroicons.
