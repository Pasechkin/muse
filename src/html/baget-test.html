📐 Схема сборки рамы (как полоска и уголок стыкуются) 
Полоска (1.webp) = ВЕРТИКАЛЬНЫЙ брусок рамы (левая сторона)
Уголок  (2.webp) = ВЕРХНИЙ ЛЕВЫЙ угол рамы

Сборка:
┌──────────┬────────────────────────────┬──────────┐
│  Уголок  │  Полоска повёрнута +90°    │  Уголок  │
│  as-is   │  (CW → горизонталь)       │  flipX   │
│  (TL)    │  TOP SIDE                  │  (TR)    │
├──────────┼────────────────────────────┼──────────┤
│ Полоска  │                            │ Полоска  │
│ as-is    │        ФОТО               │ flipX    │
│ repeat-y │                            │ repeat-y │
│ LEFT     │                            │ RIGHT    │
├──────────┼────────────────────────────┼──────────┤
│  Уголок  │  Полоска повёрнута -90°    │  Уголок  │
│  flipY   │  (CCW → горизонталь)        │  flipXY  │
│  (BL)    │  BOTTOM SIDE              │  (BR)    │
└──────────┴────────────────────────────┴──────────┘

Узор полоски обращён ВНУТРЬ (к картине).
Если не совпадает — включите «Отзеркалить полоску».
